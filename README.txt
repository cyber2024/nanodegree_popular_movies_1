Picasso requires an API key to utilise it's API.

Please open the res/values/strings.xml file:
\MovieStreamer\app\src\main\res\values\strings.xml

edit the below string resource to include your api key as the value:
<string name="picasso_api_key">ENTER YOUR PICASSO API KEY HERE</string>

Compile and run!
